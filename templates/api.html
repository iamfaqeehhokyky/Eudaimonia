Eudaimonia API Documentation

The Eudaimonia API is a powerful tool for developers to integrate mental health management features into their applications. It offers functionalities such as mood training, interventions, community support, and data privacy. By leveraging this API, developers can prioritize mental well-being, empowering users to access personalized interventions, and connect with others on their mental health journey. Join us in revolutionizing application development with the Eudaimonia API and make mental health a core focus of your app.

Get All Stress Management Resources

GET /resources

Retrieves all stress management resources.

Response:

{
  "resources": [
    {
      "id": 1,
      "title": "Resource Title",
      "description": "Resource Description",
      "link": "Resource Link"
    },
    {
      "id": 2,
      "title": "Resource Title",
      "description": "Resource Description",
      "link": "Resource Link"
    },
    ...
  ]
}

Get Stress Management Resource

GET /resources/{id}

Retrieves a specific stress management resource.

Response:

{
  "resource": {
    "id": 1,
    "title": "Resource Title",
    "description": "Resource Description",
    "link": "Resource Link"
  }
}

Create Stress Management Resource

POST /resources

Creates a new stress management resource.

Request Body:

{
  "title": "Resource Title",
  "description": "Resource Description",
  "link": "Resource Link"
}

Response:

{
  "resource": {
    "id": 1,
    "title": "Resource Title",
    "description": "Resource Description",
    "link": "Resource Link"
  }
}

Update Stress Management Resource

PUT /resources/{id}

Updates an existing stress management resource.

Request Body:

{
  "title": "Updated Resource Title",
  "description": "Updated Resource Description",
  "link": "Updated Resource Link"
}

Response:

{
  "resource": {
    "id": 1,
    "title": "Updated Resource Title",
    "description": "Updated Resource Description",
    "link": "Updated Resource Link"
  }
}

Delete Stress Management Resource

DELETE /resources/{id}

Deletes a specific stress management resource.

Response:

Status Code: 204 No Content

SDKs and Libraries

Currently, there are no SDKs or client libraries available for the Eudaimonia API. However, you can use standard HTTP libraries in your preferred programming language to interact with the API.
